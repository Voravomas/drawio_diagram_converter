# drawio_diagram_converter
1. Converts from `.json` to draw.io UML diagram<br />
2. Converts from `dict` to draw.io UML diagram<br />
Python 3.8<br />
Usage in `example.py`<br />
sample.json
```
{
    "tableName1": {
        "key1": "string",
        "key2": "integer"
    },
    "tableName2": {
        "key1": "string",
        "key2": "integer"
    },
    "tableName3": {
        "key1": 55,
        "key2": 44.4
    }
}
```
